Compiling and running the program. Type the following in the command prompt/cmd.exe or in the linux terminal.
1)javac InformationRetreivalBeta.java
2)java InformationRetreivalBeta

GUI will appear where you have to set the directory path of the dataset folder we have provided in the following way:
C:\\IRassignment\\dataset   
Beware of putting double back-slash

Now put query in the query field and click on search. Wait for some time as the document size is large. It will show you the result.

Dataset test cases
  QUERY
1)QUERY:hobbit ring
  recall:100 %
  precision:40 %

2)QUERY:rahel ring

  recall:100 %
  precision:60 %

3)QUERY:google ring rahel

  recall:100 %
  precision:80 %
